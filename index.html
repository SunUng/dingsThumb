dingsThumb

썸네일 이미지의 포지셔닝과 썸네일 박스와 이미지의 비율을 고려한 다양한 옵션을 제공합니다.

Getting started

플러그인 소개

익스플로러8에서 작동되지 않는 background-size:cover; 의 기능을 ie8에서도 사용할 수 있도록 하기 위해 개발하였습니다. 이미지의 width와 height, 위치값은 absolute, top, left 만으로 컨트롤 하므로 하위브라우저에서도 background-size:cover; 와 같은 기능을 제공할 수 있을 것으로 기대합니다. 그 외에도 썸네일 배경, 로딩효과, 위치 설정 등의 다양한 옵션을 함께 제공합니다.

사용법

  1. 1. body의 끝 부분에 .js 파일을 모두 링크합니다. jQuery를 기반으로 제작한 플러그인이므로, 제이쿼리를 먼저 링크합니다. 
    <script src="/your_path/jquery.min.js"></script>
<script src="/your_path/dingsThumb.min.js"></script>
  2. 2. jQuery와 plugin 파일이 연결되었다면, 플러그인을 호출합니다. 플러그인을 적용할 엘리먼트에 플러그인 함수를 호출하여 사용합니다. 사용자 정의 및 옵션 사용법은 아래의 데모를 참조하세요. 
    $(document).ready(function(){
  $(".elements").dingsThumb();
});

        var obj = box.find('img');

        var boxRatio = boxHeight / boxWidth;
        var imgHeight = obj.height(), imgWidth = obj.width(), imgRatio = imgHeight / imgWidth;
        var referCalc = (imgRatio / boxRatio) > 1, refer = cover === true ? referCalc : !referCalc;
        var valuesSize = (function() {
          var value = [];
          if (refer === true) {
            value[0] = '100%';
            value[1] = 'auto';
          } else {
            value[0] = 'auto';
            value[1] = '100%';
          }

        })();
        obj.css({
          'position':'absolute',
          'display':'inline-block',
          'width':valuesSize[0],
          'height':valuesSize[1]
        });
        var modifiedImgW = obj.width(), modifiedImgH = obj.height();
        var valuesPosition = (function() {
          var value = [];
          if (horizon === 'left') {
            value[0] = 0;
          } else if (horizon === 'right') {
            value[0] = boxWidth - modifiedImgW;
          } else  {
            value[0] = (boxWidth / 2) - (modifiedImgW / 2);
          }
          if (vertical === 'top') {
            value[1] = 0;
          } else if (vertical === 'bottom') {

          } else {

          }

        })();

          'top':valuesPosition[1],
          'left':valuesPosition[0]
        });
      }
      function thumbNoloading() {
        $('').attr("src", src).load(function() {
          $(this).appendTo(thumb).siblings('img').remove();
          thumbAdjust();
        });
      }
      function thumbLoading() {
        $('').attr("src", src).load(function() {
          $(this).appendTo(thumb).fadeIn('slow').siblings('img').remove();
          thumbAdjust();
        });
      }
      if (loaded === false) {
        var imageObj = loading === false ? thumbNoloading() : thumbLoading();
        var source = $('
').css({'position' : 'relative', 'overflow' : 'hidden', 'background' : bg, 'width' : boxWidth, 'height' : boxHeight});
        box.wrapInner(source);
        var thumb = box.find('.thumb-wrapper');

      }
      $(document).ready(thumbAdjust);
      $(window).on('load resize', thumbAdjust);
    });
  };
  $.fn.dingsThumb.defaults = {
      cover : true,
      loading : false,
      background : '#fff',
      vertical : 'center',

Options

  • cover

    타입 : Boolean

    기본 : true

    true 사용시 상자와 이미지의 크기, 비율에 상관없이 언제나 상자에 가득찬 모습으로 보입니다. 이 때, 비율 상 상자에 넘치는 이미지는 가려져 보입니다. false 사용시 이미지는 잘리는 영역이 없이 이미지의 모든 부분이 상자 안에 나타납니다.

  • loading

    타입 : Boolean

    기본 : false

    loading 옵션을 true로 설정하면 이미지가 로드될 때 fadeIn 애니메이션 효과가 발생합니다. 다만, 플러그인은 DOM이 모두 렌더링 한 후에 작동하므로 이미지가 반짝 보여지는 일이 없도록 하려면, 이 옵션을 사용하기 전에 CSS를 통해 미리 이미지를 display:none;으로 설정하는 것이 좋습니다.

  • background

    타입 : String

    기본 : #fff

    cover 옵션이 false일 때, 상자와 이미지의 비율이 동일하지 않을 때 필연적으로 상자와 이미지 사이에 여백이 생깁니다. 그 여백의 색상을 조절할 필요가 있다면 이 옵션을 사용합니다. 브라우저가 읽을 수 있는 헥스코드나 색상명, RGB 등을 문자(string)의 형태로 사용합니다.

  • vertical

    타입 : String

    기본 : center

    이미지가 수직 방향으로 정렬될 위치를 선정합니다. 상단으로는 top, 중간으로는 center, 하단으로는 bottom을 사용할 수 있습니다.

  • horizon

    타입 : String

    기본 : center

    이미지가 수평 방향으로 정렬될 위치를 선정합니다. 좌측으로는 left, 중간으로는 center, 우측으로는 bottom을 사용할 수 있습니다.

Demos

  • default
    Option 기본값 
    $('.demo1').dingsThumb();
  • cover
    cover : false
    cover 옵션을 끔으로써 이미지가 모든 부분이 상자 안으로 들어갑니다. 
    $('.demo2').dingsThumb({
    cover : false
});
  • loading
    loading : true
    이미지가 로드될 때 fadeIn 애니메이션 효과가 발생합니다. 
    $('.demo3').dingsThumb({
    loading : true
});
  • background
    background : '#a3d553'
    상자와 이미지의 비율이 동일하지 않아서 나타나는 여백에 배경색을 입힙니다. 헥스코드 '#a3d553'를 입력해 초록계열 배경을 만듭니다. 이 에제에서는 옵션의 효과를 잘 표현하기 위해 cover 옵션을 false로 하여 여백이 나타나도록 했습니다. 
    $('.demo4').dingsThumb({
    cover : false,
    background : '#a3d553'
});
  • vertical
    vertical : 'bottom'
    이미지의 수직방향 정렬을 bottom, 즉 아래로 설정합니다. 이렇게하면 이미지의 아랫부분을 중심으로 나타냅니다. 비율에 따라 cover 옵션을 사용할 때나 사용하지 않을 때 모두 작동합니다. 이 예제에서는 옵션의 효과를 잘 표현하기 위해 cover 옵션을 false로 하여 여백이 나타나도록 했고 background 옵션으로 여백 배경색을 설정했습니다. 
    $('.demo5').dingsThumb({
    background : '#e85764',
    cover : false,
    vertical : 'bottom'
});
  • horizon
    horizon : 'right'
    이미지의 수평방향 정렬을 right, 즉 오른쪽으로 설정합니다. 이렇게하면 이미지의 오른쪽 부분을 중심으로 나타냅니다. 비율에 따라 cover 옵션을 사용할 때나 사용하지 않을 때 모두 작동합니다. 이 예제에서는 옵션의 효과를 잘 표현하기 위해 cover 옵션을 false로 하여 여백이 나타나도록 했고 background 옵션으로 여백 배경색을 설정했습니다. 
    $('.demo6').dingsThumb({
    background : '#fdd030',
    cover : false,
    horizon : 'right'
});
